# Proyecto: Generador Seguro de Contraseñas

## Introducción

En la actualidad, la seguridad informática se ha convertido en una necesidad debido al creciente uso de plataformas digitales para actividades personales, académicas y laborales. Una de las principales recomendaciones para proteger la información es utilizar contraseñas fuertes y difíciles de adivinar. Sin embargo, muchas personas continúan empleando claves débiles o repetidas en diferentes servicios, lo que incrementa el riesgo de sufrir ataques informáticos.

Con el propósito de contribuir a una mejor seguridad digital, se desarrolló un Generador Seguro de Contraseñas utilizando Python. Este sistema permite crear contraseñas aleatorias con diferentes características de seguridad, ofreciendo una herramienta práctica, sencilla y eficiente para los usuarios.

## Descripción del problema

Uno de los problemas más frecuentes en la seguridad informática es el uso de contraseñas débiles, cortas o fáciles de adivinar. Muchas personas utilizan información personal, fechas de nacimiento o palabras comunes, lo que facilita que ciberdelincuentes puedan acceder a sus cuentas mediante ataques de fuerza bruta o diccionario.

El proyecto busca solucionar esta problemática permitiendo generar contraseñas robustas con diferentes niveles de complejidad, reduciendo así la probabilidad de accesos no autorizados y promoviendo buenas prácticas de ciberseguridad.

## Explicación del sistema desarrollado

El sistema desarrollado consiste en una aplicación capaz de generar contraseñas seguras de manera automática.

Su funcionamiento comienza solicitando al usuario la longitud deseada para la contraseña. Posteriormente, el programa permite generar una combinación aleatoria de letras mayúsculas, letras minúsculas, números y caracteres especiales, garantizando una mayor seguridad.

El sistema también valida la información ingresada para evitar errores y ofrece la posibilidad de generar nuevas contraseñas cuando el usuario lo desee.

Entre sus principales funcionalidades se encuentran:

* Generación automática de contraseñas.
* Personalización de la longitud.
* Inclusión de distintos tipos de caracteres.
* Validación de la entrada del usuario.
* Interfaz sencilla y fácil de utilizar.
* Posibilidad de generar múltiples contraseñas durante una misma ejecución.
